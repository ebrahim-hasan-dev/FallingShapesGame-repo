# Falling Shapes Game

A lightweight arcade-style game built with Python. This project is designed to test reflexes and provides a clean, documented example of basic game logic and GUI implementation.

## Features
* **Dynamic Gameplay:** Shapes fall at varying positions and speeds.
* **Score System:** Real-time point tracking based on successful catches.
* **Clean Architecture:** Organized Python source files for easy reading and maintenance.
* **Portable:** Can be converted to a standalone executable for Windows.

## Tech Stack
* **Language:** Python 3.x
* **IDE:** Visual Studio Code
* **Deployment:** PyInstaller (for .exe creation)

## Installation & Setup
To run this project locally, ensure you have Python installed, then follow these steps:

1. Clone the repository:
   git clone [https://github.com/ebrahim-hasan-dev/FallingShapesGame-repo.git]

2. Navigate to the project directory:
   cd FallingShapesGame-repo

3. Run the main file:
   python main.py

## How to Play
* **Move:** Use the Left and Right Arrow keys on your keyboard.
* **Goal:** Catch as many falling shapes as possible to increase your score.
* **Game Over:** Avoid picking up white frogs.
